## Features
- Fetches jokes from an external API
- Category selection (Random, General, Programming, Knock-Knock)
- Generate new joke on button click
- Loading indicator while fetching data
- Clean card-based UI
- Fully responsive design

## Technologies Used

- React.js
- JavaScript (ES6+)
- CSS

## API Used

Official Joke API : https://official-joke-api.appspot.com/

## How It Works

- On first load, a joke is fetched using `useEffect()`
- When the user selects a category, a new joke is fetched automatically
- Clicking **Get New Joke** fetches another joke from the selected category
